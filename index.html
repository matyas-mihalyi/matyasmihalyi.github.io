Interval timer

I made this with vanilla javascript, so it helped me reinforce the basics, especially arrays, objects, classes and rendering the DOM.

I’ve tried a few timer apps for HIIT training, but I always found them hard to set up. With this app I’d like to introduce a more user friendly approach, where users can reorganise intervals via drag & drop and add repeat to multiple intervals after they’ve been set.

Development is currently halted, because I started to learn libraries, frameworks and NodeJS.
