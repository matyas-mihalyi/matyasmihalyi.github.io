Interval timer

I’ve tried a few timer apps for HIIT training, but I always found them hard to set up.

With this app I’d like to introduce a more user friendly approach, where users can reorganise intervals via drag & drop and add repeat to multiple intervals after they’ve been set.

This is a work in progress project, currently optimised for mobile only. I’m still adding/testing out some functions before finalising the UI. I usually update it on the weekends.

Dice roller
